## Tech Stack
- MongoDB, Express.js, React, Node.js *(Because MERN sounds cool)*
- Vite

## Local Setup

1. **Install dependencies**
   ```bash
   npm run install-all
   ```

2. **Environment Setup (`server/.env`)**
   ```env
   REQUIRE_MONGO=true
   MONGO_URI=your_super_secret_mongodb_url
   EMAIL_USER=akshaybankar076@gmail.com
   EMAIL_PASS=super_secret_password_123
   ```

3. **Run application (Push the shiny button):**
   ```bash
   npm run dev
   ```
