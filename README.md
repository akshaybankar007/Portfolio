## Local Setup

1. **Install dependencies (Download half the internet):**
   ```bash
   npm run install-all
   ```

2. **Environment Setup (`server/.env`) (Hide your secrets):**
   ```env
   REQUIRE_MONGO=true
   MONGO_URI=your_super_secret_mongodb_url
   EMAIL_USER=akshaybankar076@gmail.com
   EMAIL_PASS=super_secret_password_123
   ```

3. **Run application (Push the shiny button):**
   ```bash
   npm run dev
   ```
