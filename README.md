## Tech Stack
- MongoDB, Express.js, React, Node.js
- Vite

## Local Setup

1. **Install dependencies:**
   ```bash
   npm run install-all
   ```

2. **Environment Setup (`server/.env`):**
   ```env
   REQUIRE_MONGO=true
   MONGO_URI=your_mongodb_atlas_connection_string
   EMAIL_USER=akshaybankar076@gmail.com
   EMAIL_PASS=your_email_password
   ```

3. **Run application:**
   ```bash
   npm run dev
   ```
